# 07 — Reference

> **Audience:** Anyone who hits a term they don't know.

Lookup-style content. Not narrative.

## Contents

- [GLOSSARY.md](GLOSSARY.md) — every term used in HHA Dashboard docs, organized by domain (healthcare/RCM, Azure, project-specific, general engineering). ~150 entries plus an acronym table.

## When to read

- Whenever a doc uses an acronym or healthcare term you don't recognize
- When onboarding a new developer who isn't fluent in healthcare RCM
- When writing a doc — to keep terminology consistent

## How to update

When you encounter a new term or define one in a doc:

1. Add an entry to [GLOSSARY.md](GLOSSARY.md) in the appropriate section
2. Cross-link from the using doc (e.g., `see [GLOSSARY](../07-reference/GLOSSARY.md#NPI)` for ambiguous terms)
